# Claude Code 첫 프롬프트 (복사해서 붙여넣기)

이 프로젝트를 처음 Claude Code에서 열고 나서 채팅창에 붙여넣을 만한 시작 프롬프트입니다. 상황에 맞게 골라 쓰세요.

---

## 옵션 A — 환경 세팅부터 첫 실행까지 함께 가기

```
이 프로젝트는 사티의 피아노 음악을 19세기 후반 파리 양식 시각 예술로 번역하는 교차 모달 AI 파이프라인이야.
CLAUDE.md를 먼저 읽고 전체 구조를 파악해 줘.

지금 내 상황:
- OS: Windows 11
- GPU: [네 GPU 모델, 예: RTX 3060 12GB / 없음 / Colab 사용 예정]
- Python: [버전 또는 "아직 없음"]

다음 단계로 안내해 줘:
1. 환경 세팅 (uv 또는 conda 추천 + 명령)
2. 사티 mp3 어디서 받을지 (data/audio/)
3. 첫 실행 명령
4. 결과 확인 방법
```

## Claude Code 사용 팁

- **CLAUDE.md는 자동으로 컨텍스트에 포함됩니다.** 매번 따로 읽어 달라고 할 필요 없지만, 처음에는 명시적으로 시키는 게 안전합니다.
- **`/init` 명령**: Claude Code가 프로젝트 구조를 자동으로 분석하고 CLAUDE.md를 업데이트해 줍니다. 이미 만들어져 있으니 지금은 불필요.
- **파일 첨부**: 음원 분석 결과나 생성 이미지를 채팅창에 드래그하면 Claude Code가 직접 봅니다 (이미지·텍스트 모두).
- **`#` 컨텍스트 메시지**: 작업 중간에 "지금 vexations.png 결과가 너무 밝아"처럼 적으면 Claude Code가 그 맥락에서 다음 액션을 잡습니다.
